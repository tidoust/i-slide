Demo of the <i-slide> Web component

<i-slide> is a Web component that allows embedding HTML and PDF slides inside a Web page.

To use it, wrap <span is=i-slide>...</span> around a link to a slide.

It works with the Shower HTML slides engine.

<span is=i-slide><a href='http://shower.github.io/shower/?full#2'>Shower slide 2</a></span>

allows to embed the second slide of the referenced slideset:

Shower slide 2

It also works with the b6+ HTML slides engine.

<span is=i-slide><a href='https://www.w3.org/Talks/Tools/b6plus/?full#3'>B6+ slide 3</a></span>

allows to embed the third slide of the referenced slideset:

B6+ slide 3

It also works with the PDF slides.

<span is=i-slide><a href='https://www.w3.org/2020/Talks/mlws/ba-media.pdf#page=5'>PDF slide 5</a></span>

allows to embed the fifth slide of the referenced slideset:

PDF slide 5

It is also possble to use <i-slide> as a tag. But note that this means there will be no link to the slide if JavaScript is turned off or it the page is used elsewhere than in a browser.
